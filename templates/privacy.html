개인정보 처리방침

최종 업데이트: 2025-01-01

QuickPDFSum(이하 “서비스”)는 이용자의 개인정보를 중요하게 생각하며, 관련 법령을 준수합니다. 본 개인정보 처리방침은 서비스 이용 시 수집·이용되는 정보와 보호 방침에 대해 설명합니다.

1. 수집하는 개인정보의 항목

서비스는 원칙적으로 이용자의 직접 식별 가능한 개인정보를 저장하지 않으며, 다음 항목을 일시적으로 처리할 수 있습니다.

  • 접속 IP 주소
  • 브라우저/디바이스 정보(User-Agent)
  • 요청 횟수 및 요청 시간(요청 제한 및 악용 방지 목적)
  • 업로드된 PDF 파일 및 해당 PDF에서 추출된 텍스트(OCR 결과 포함)

PDF 내용은 AI 기반 요약을 위해 OpenAI API로 전송될 수 있으며, OpenAI는 API 입력 데이터를 모델 학습에 사용하지 않습니다.

2. 개인정보의 이용 목적

서비스는 수집된 정보를 다음 목적 범위 내에서만 이용합니다.

  • PDF 문서 요약 기능 제공
  • 비정상적·과도한 사용 방지(IP 기반 요청 제한)
  • 서비스 품질 개선 및 오류·장애 분석
  • 보안 및 안정성 확보

서비스는 수집된 정보를 타겟 광고나 이용자 프로파일링에 사용하지 않습니다.

3. 개인정보의 보관 및 파기

  • PDF 파일: 요약 처리 완료 후 즉시 서버에서 삭제되며, 데이터베이스 등에 장기 보관되지 않습니다.
  • 로그 데이터: 서버 운영 환경에 따라 약 7~30일 내 자동 삭제됩니다.
  • 요청 횟수 정보: 일 단위로 초기화되며 장기간 보관하지 않습니다.

4. 개인정보의 제3자 제공

서비스는 다음의 경우를 제외하고 제3자에게 정보를 제공하지 않습니다.

  • 이용자가 사전에 동의한 경우
  • 법령에 의하여 요청이 있는 경우
  • AI 요약 기능 제공을 위해 OpenAI API로 텍스트를 전송하는 경우

5. 개인정보의 안전성 확보 조치

서비스는 개인정보 보호를 위하여 다음과 같은 조치를 취합니다.

  • HTTPS/TLS를 통한 통신 구간 암호화
  • 파일 저장 최소화 및 처리 완료 후 즉시 삭제
  • 서버 방화벽 및 접근 제어 설정
  • 보안 패치 및 라이브러리 업데이트 적용

6. 이용자의 권리

서비스는 개인정보를 장기간 보관하지 않으므로, 열람·정정·삭제 요청 시 로그 및 시스템 구조 상 가능한 범위 내에서 처리합니다. 데이터 처리에 대한 문의가 있는 경우 아래 연락처로 문의할 수 있습니다.

7. 쿠키(Cookie)의 사용

서비스는 세션 관리 및 요청 제한 등을 위해 최소한의 기술적 쿠키를 사용할 수 있습니다. 광고·추적 목적의 쿠키는 사용하지 않습니다.
